--- layout: guide guideName: Home Assistant Green LEDs ---

Home Assistant Green has 3 LEDs.

  • White LED: Power LED
  • Green LED: Activity LED
  • Yellow LED: Home Assistant Green status LED

Power LED (white)

The power LED indicates whether the CPU is powered. If the LED is off, check the power supply.

Activity LED (green)

The activity LED indicates various firmware and boot loader states at startup. Once Home Assistant OS has been started, the green LED indicates activity. Activity is defined as disk (eMMC) or any CPU load.

Status LED (yellow)

Home Assistant OS

Pattern Description
Off System is in EEPROM firmware or boot loader mode.
Heartbeat Home Assistant OS is running.
