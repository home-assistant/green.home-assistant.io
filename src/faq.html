- title: > Why is there an optional CR2032 battery? Do I need to add one? content: >

  • The battery serves as a backup power supply for the real-time clock.
  • You need to install a battery only if you are planning to boot the Home Assistant Green in an environment where there is no internet connection during startup. Then the clock cannot synchronize.
